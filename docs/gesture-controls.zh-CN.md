# WaveDeck 模式、快捷键与手势说明

本文档描述 `v0.1.0` 当前公开版本中的控制方式。

## 第一条规则

手势控制只有在以下条件同时满足时才会生效：

- 已经导入 PPT 项目
- 当前处于手势模式
- 手势运行时没有被锁定

按 `Ctrl+Alt+M` 可以进入手势模式。

## 模式说明

### PPT 预览

适合通过缩略图快速找页。

- 左侧缩略图栏：直接选页
- 双击中央页面：进入放映模式
- `Ctrl+Alt+M`：切换到手势模式

### PPT 放映

适合只看当前页的干净展示状态。

- 单击：下一页
- 双击：返回预览
- `Esc`：返回预览
- `Ctrl+Alt+M`：切换到手势模式

### 手势滚筒

适合在相邻页面之间做空间化浏览。

- 中央页是当前页
- 两侧页面保留上下文
- 鼠标拖动 / 滚轮：旋转滚筒
- `Ctrl+Alt+M`：返回上一次使用的 PPT 子模式

### 手势单页舞台

适合放大、平移、切页和激光指示。

- 底部控制层会显示缩放控件
- `Esc`：返回滚筒
- `Ctrl+Alt+M`：返回 PPT 模式

## 快捷键总表

| 快捷键 | 功能 |
| --- | --- |
| `Ctrl+O` | 打开 PowerPoint 文件 |
| `Ctrl+Alt+M` | 在 PPT 模式和手势模式之间切换 |
| `PageDown`、`Right`、`Space`、`N` | 下一页 |
| `PageUp`、`Left`、`P` | 上一页 |
| `Home` | 第一页 |
| `End` | 最后一页 |
| `F11` | 进入或退出全屏 |
| `Esc` | 退出全屏，或从手势单页舞台返回滚筒 |

## 手势映射总表

下表描述的是当前 `gesture/controller.py` 中真实实现的公开行为。

| 手势 | 生效位置 | 功能 |
| --- | --- | --- |
| 单手握拳并保持约 `650 ms` | 手势模式 | 锁定手势控制 |
| 锁定后出现任意非握拳手势 | 手势模式 | 自动解锁 |
| 张开手掌，向左挥动 | 手势滚筒 / 手势单页舞台 | 下一页 |
| 张开手掌，向右挥动 | 手势滚筒 / 手势单页舞台 | 上一页 |
| `OK` 手势保持约 `90 ms` | 手势滚筒 | 进入当前页的单页舞台 |
| `OK` 手势保持约 `90 ms` | 手势单页舞台 | 返回手势滚筒 |
| 仅伸出右手食指 | 手势单页舞台 | 平移页面 |
| 双手张开并向两侧分开 | 手势单页舞台 | 放大 |
| 双手张开并向中间靠拢 | 手势单页舞台 | 缩小 |
| 食指和中指伸出，无名指和小指收起 | 手势单页舞台 | 激光指示 |

## 手势细节

### 1. 锁定 / 解锁

- 单手握拳保持一小段时间后，会暂停手势命令。
- 这适合你想把手留在镜头里，但暂时不希望继续触发动作的时候。
- 恢复方式很简单：张开手，或者换成任意非握拳姿态。

### 2. 挥手翻页

- WaveDeck 会把横向张掌挥动识别为翻页动作。
- 向左挥：下一页
- 向右挥：上一页
- 当前翻页冷却大约是 `450 ms`，所以不会因为一个动作连续连翻很多页。

### 3. OK 手势

`OK` 手势在手势模式里承担“内部模式切换”作用：

- 在滚筒里：打开当前页，进入单页舞台
- 在单页舞台里：返回滚筒

它和 `Ctrl+Alt+M` 不一样。

- `OK`：在手势模式内部切换滚筒和单页
- `Ctrl+Alt+M`：在 PPT 模式和手势模式之间切换

### 4. 平移

- 平移只在手势单页舞台中可用。
- 当前实现要求“仅右手食指伸出”的姿态。
- 很小的抖动会被忽略，避免页面来回晃。

### 5. 缩放

- 缩放只在手势单页舞台中可用。
- 两只手都需要是张开的手掌。
- 控制器会先等待一个很短的激活窗口，再判断是否真的要进入缩放。
- 进入后：
  双手分开就是放大
  双手靠近就是缩小

### 6. 激光指示

- 激光指示只在手势单页舞台中可用。
- 当前姿态要求：
  食指伸出
  中指伸出
  无名指收起
  小指收起

激光点本身带平滑处理，也能容忍很短时间的识别抖动。

## 推荐上手顺序

如果你第一次尝试手势模式，建议按这个顺序来：

1. 先用 `Ctrl+Alt+M` 进入手势模式
2. 先练张掌左右挥动翻页
3. 再练 `OK` 手势，在滚筒和单页之间切换
4. 等单页舞台稳定以后，再尝试双手缩放
5. 需要暂停时，用握拳锁定

## 常见问题

### 手势完全没反应

按顺序检查：

1. 是否已经打开 PPT
2. 当前是否真的在手势模式，而不是 PPT 预览 / 放映
3. 当前是否被握拳锁定

### 握拳之后像“死掉了一样”

大概率不是坏了，而是进入了锁定状态。

- 张开手
- 或者换成任意非握拳姿态

当前版本会在识别到非握拳后自动解锁。

### 双手缩放不容易触发

- 两只手都尽量保持清晰的张掌
- 先稳定出现在画面里一小会儿
- 然后再明确地向两边拉开或向中间靠拢

### 页面在移动，但没有翻页

这通常说明你当前在手势单页舞台里，并且用的是更接近平移的姿态。

- 想翻页时，使用张掌横向挥动
- 如果想先回到滚筒，再用 `OK` 手势切回去

## 说明

本文档描述的是当前公开行为。手势体验还在继续优化，所以阈值和识别细节后续可能继续调整。
