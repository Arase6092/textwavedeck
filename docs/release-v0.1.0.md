# v0.1.0 - Public Launch

## What Is Included

- PowerPoint `.ppt` and `.pptx` import on Windows
- Microsoft PowerPoint COM based PNG export
- 4K slide cache with atomic replacement
- Thumbnail preview workspace
- Clean slideshow workspace
- Five-page theatre carousel
- Single-slide stage with zoom and pan
- Keyboard shortcuts, fullscreen, and reduced-motion support
- Chinese in-app status, warnings, and logs

## Who Should Try It

- Windows users who review decks frequently
- Presenters who want a more cinematic slide browser
- Developers building gesture-first presentation workflows on top of a stable local slide pipeline

## Current Limitations

- Windows only
- Microsoft PowerPoint is required
- Source release only in `v0.1.0`
- Gesture runtime is still evolving and is not the focus of the public screenshots

## Suggested Repository Description

WaveDeck is a local-first Windows PowerPoint stage with 4K slide export, thumbnail preview, cinematic carousel navigation, and a gesture-ready architecture.
