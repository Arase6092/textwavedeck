# Security Policy

## Supported Versions

The latest tagged release is the supported public version.

## Reporting

Please open a private security report through GitHub Security Advisories or contact the maintainer directly before publishing details.

## Privacy Boundaries

- Do not submit private decks, tokens, cookies, customer information, or unpublished material in issues.
- Do not include screenshots that expose camera footage or sensitive presentation content unless explicitly redacted.
- Public logs and bug reports should describe behavior, not reveal deck contents.
