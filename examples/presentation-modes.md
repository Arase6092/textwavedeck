# Presentation Modes

## Preview Mode

Best for scanning a deck, checking sequence, and jumping via thumbnails.

## Slideshow Mode

Best for focusing on one slide at a time with PowerPoint-style keyboard navigation.

## Theatre Carousel

Best for spatially browsing nearby slides while keeping the current page centered.

## Single-Slide Stage

Best for zooming into details, panning around a dense slide, or swiping between neighboring pages inside the theatre workflow.
