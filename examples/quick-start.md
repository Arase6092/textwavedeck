# Quick Start Example

## Scenario

You have a large PowerPoint deck and want to review it quickly without staying inside the full PowerPoint editing interface.

## Flow

1. Launch WaveDeck.
2. Press `Ctrl+O`.
3. Select a `.ppt` or `.pptx` file.
4. Wait for the first local export to finish.
5. Browse with the left thumbnail rail.
6. Double-click the main slide to enter slideshow mode.
7. Press `Ctrl+Alt+M` to switch into the theatre carousel when you want broader page context.

## Outcome

The next time you open the same deck without changes, WaveDeck reuses the local cache and reopens much faster.
